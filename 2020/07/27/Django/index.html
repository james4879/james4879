Django

Author: Same! Date: July 27, 2020  11:21:20

Django是python最受歡迎的兩大web框架之一 (另一個是flask), 相較於flask, Django功能很多, 也相對肥大. 這對於初學者來說較友善, 因為它對於常見的web app功能都有完整的支援; 如果經驗老道, 很清楚自己要什麼, 那flask是個不錯的選擇.

先弄好以下：python3, python3 的虛擬環境, 某款python的ide (以下以PyCharm為準), 這部分就不多說了, 自己找資料吧!

搞定之後在pycharm開一個新的project, 類型選django, 其它目錄跟路徑選項就依照個人喜好去設定…應該沒有什麼要改的, 取個名字吧, 例如 myProjectName之類的

開好之後在iTerm2試試看 bash python manage.py runserver, 如果一切順利的話應該會成功啟動. 這是用Django內建一個很陽春的開發用web server, 真的生產環境不會這樣搞. ctrl - C 可以關掉這個server

接著要建立一個”app”, 可以想像成一個project下面切割出來些功能的獨立個體, 可以在不同project間重複利用: 同樣在iTerm2 打 bash python manage.py startapp genericSignup, 顯然名稱自己可以取合適的..這邊我們想做個簡單的活動報名網站, 所以取這名字
